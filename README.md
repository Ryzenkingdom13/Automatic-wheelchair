# Automatic Voice Controlled Wheelchair

## Project Overview
This project enables physically challenged individuals to control a wheelchair using voice commands.

## Features
- Voice command recognition
- Forward, Backward, Left, Right movement
- Motor driver control
- Embedded system implementation
- Obstacle detection
- GPS tracking
- speed control

## Technologies Used
- PIC16F887
- Embedded C
- MPLAB IDE
- Proteus
- Motor Driver

## Future Enhancements
- Mobile app integration
